== Description ==

This plugin allows you to add a wallet balance functionality to user profiles in WordPress. Users can have a wallet balance which can be used for various purposes, such as making purchases, paying bills, etc.

== Features ==

- Add wallet balance field to user profile.
- Display wallet balance on the user's account page.
- Administrators can manage users' wallet balances.
- Wallet balance is shown in the list of users in the admin panel.
- Custom column with wallet balance information in the users list.
- Ensures user has a default wallet balance of 0.

== Installation ==

1. Upload the `wallet-credit-payment-gateway-for-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings from the WordPress admin panel.

== Frequently Asked Questions ==

= How do I add wallet balance to user profiles? =

The wallet balance field will be automatically added to user profiles once the plugin is activated. Administrators can manage users' wallet balances from the WordPress admin panel.

= How can users view their wallet balance? =

Users can view their wallet balance on their account page. The balance will be displayed along with other account information.
